# TrustJobAI 🔍

> AI-powered Job Scam Detector for Indian job seekers

## What is TrustJobAI?
TrustJobAI helps Indian job seekers instantly detect fake and scam job postings using Groq AI. Paste any job description or upload a screenshot — get an instant Trust Score with detailed analysis.

## Features
- 🔍 AI-powered Scam Detection (Score 0-100)
- 📋 Auto Job Details Extraction (Company, Role, Salary, Contact)
- 🏢 Company Intelligence (DuckDuckGo + Google + LinkedIn + MCA)
- 📸 Screenshot Upload & Analysis
- 🕒 Analysis History (localStorage)

## Tech Stack
- HTML + CSS + JavaScript
- Groq Llama 3.3 70B (Text Analysis)
- Groq Llama 4 Scout (Image Analysis)
- DuckDuckGo API (Company Intelligence)
